🎨 Frontend Slides

🎨 Overview

Create beautiful, animation-rich presentations without design skills

What is Frontend Slides?

Frontend Slides is a Claude Code skill designed for anyone who wants to create stunning web presentations without needing to know CSS or JavaScript. It uses a "show, don't tell" approach—instead of describing your aesthetic preferences, you pick from generated visual previews.

Why Choose Frontend Slides?

  • No design experience needed—AI-powered visual recommendations
  • Single HTML file output—no build tools, frameworks, or dependencies
  • Convert existing PowerPoint files directly to web format
  • Responsive, accessible code you can customize
  • Deploy to live URLs or export to PDF in seconds

Perfect For

Pitch decks, product launches, conference talks, educational content, portfolio showcases, and any presentation where you want to impress without the learning curve.

✨ Key Features

Everything you need to create professional presentations

Production Quality

🚀 Getting Started

Three ways to get Frontend Slides up and running

Option 1: Plugin Marketplace (Recommended)

The easiest way to get started. Just run two commands in Claude Code:

/plugin marketplace add zarazhangrui/frontend-slides /plugin install frontend-slides@frontend-slides

Then use it by typing /frontend-slides in Claude Code.

Option 2: Manual Installation

Copy skill files to your Claude Code skills directory:

mkdir -p ~/.claude/skills/frontend-slides/scripts cp SKILL.md STYLE_PRESETS.md viewport-base.css ~/.claude/skills/frontend-slides/ cp scripts/extract-pptx.py ~/.claude/skills/frontend-slides/scripts/

Option 3: Clone from GitHub

Get the latest development version:

git clone https://github.com/zarazhangrui/frontend-slides.git ~/.claude/skills/frontend-slides

Create Your First Presentation

  1. Type /frontend-slides in Claude Code
  2. Describe your content (topic, slides, images)
  3. Tell us the vibe (impressed, excited, calm, creative, etc.)
  4. Review and pick your favorite visual style
  5. Watch as we generate your complete presentation
  6. Open it in your browser and start presenting!

Requirements

  • Claude Code CLI installed
  • For PPT conversion: Python with python-pptx library
  • For URL deployment: Node.js + Vercel account (free)
  • For PDF export: Node.js (Playwright installs automatically)

🌐 Sharing & Export Options

Multiple ways to share your presentations with the world

Deploy to a Live URL

After creating a presentation, deploy it to a permanent, shareable URL that works on any device:

bash scripts/deploy.sh ./my-deck/ # or bash scripts/deploy.sh ./presentation.html

Uses Vercel's free tier. The script walks you through signup and login if it's your first time. Your presentation is instantly live and shareable.

Export to PDF

Convert your slides to a static PDF for email, Slack, Notion, or printing:

bash scripts/export-pdf.sh ./my-deck/index.html bash scripts/export-pdf.sh ./presentation.html ./output.pdf

Uses Playwright to screenshot each slide at 1920×1080 and combine into a PDF. Installs automatically if needed. Note: Animations are not preserved in the PDF version.

Share as HTML

Since each presentation is a single HTML file, you can:

  • Email it directly—works in any email attachment
  • Host on any server—no special configuration needed
  • Put on Google Drive or Dropbox—opens in any browser
  • Upload to GitHub and share the link
  • No dependencies means it works forever, offline too

Collaboration & Feedback

Share the live URL or PDF with your team. Since presentations are fully responsive, they look perfect on phones and tablets. Feedback can be collected through comments, or you can iterate quickly by regenerating with updated content.

💭 Our Philosophy

The beliefs that shaped Frontend Slides

1. You Don't Need to Be a Designer

You just need to react to what you see. Beautiful design doesn't require a degree—it requires iteration and feedback. We generate options, you pick the one that feels right. That's the whole process.

2. Dependencies Are Technical Debt

A single HTML file will work in 10 years. A React project from 2019? You'll spend more time upgrading dependencies than actually using it. We chose simplicity and longevity over trendy frameworks.

3. Generic Is Forgettable

Every presentation should feel custom-crafted, not template-generated. Our 12 styles are distinctive, opinionated, and designed to stand out. No purple gradients unless you ask for it.

4. Comments Are Kindness

Code should explain itself—to future-you, to collaborators, to anyone who opens it. When we generate your HTML, it includes clear, thoughtful comments so you can customize with confidence.

5. Show, Don't Tell

Rather than asking you to describe your aesthetic or download a 50-page template guide, we show you visual options. Pick what you like. Learn by seeing, not reading.

Inspired By

Frontend Slides was born from the "Vibe Coding" philosophy—the idea that you don't need to be a traditional software engineer to build beautiful things. You need judgment, iteration, and tools that get out of your way.
